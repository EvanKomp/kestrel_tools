Submit Neuralplexer Job

Instructions:

Submit a job to the Neuralplexer server. The job requires a CSV file with the following columns:

  • protein_seq (required): Amino acid sequence of the protein to be analyzed.
  • smiles (required): SMILES representation of the ligand to be docked.
  • pdb (optional): PDB file with the protein structure.

Optionally, you can provide a ZIP file with PDB files for the proteins in the CSV file.
